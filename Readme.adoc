= **Wireguard Installer**

== **Info**

This script wasn't tested so far, but can be the first. :)

== **Requirements:**

You need to install the Wireguard package manually.

Also you need to have systemd.

== **Install:**

make script.sh executable and run sudo or as root ```./script.sh install```

== **Add a client**

```
sudo ./script.sh add
```
